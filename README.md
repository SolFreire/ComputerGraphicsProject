# Modelagem de Sólidos 3D – Projeto Final

Este repositório contém a implementação completa do projeto de Modelagem e Visualização 3D:
- Cubo parametrizado
- Toro parametrizado
- Cano curvado com curva Bézier/Hermite
- Geração de malha (triângulos)
- Construção de cena sem interseções
- Transformações (escala, rotação, translação)
- Sistema de coordenadas da câmera
- Projeção em perspectiva
- Rasterização em múltiplas resoluções
